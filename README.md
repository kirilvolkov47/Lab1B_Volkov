## Використаний алгоритм

Застосовано алгоритм **модифікованих жорданових виключень**. Він схожий на звичайні ЖВ з Part A, але відрізняється тим що при виконанні кроку знак змінюють елементи розв'язувального стовпця, а не рядка. Решта алгоритму залишається такою ж: формула прямокутника, ділення на розв'язувальний елемент, обмін змінних.

На основі цього алгоритму реалізовано:

- **Пошук опорного розв'язку** — перевірка стовпця вільних членів на від'ємні елементи та виконання кроків МЖВ поки всі значення не стануть невід'ємними
- **Пошук оптимального розв'язку** — перевірка Z-рядка на від'ємні елементи та виконання кроків МЖВ поки всі елементи Z-рядка не стануть невід'ємними

### Алгоритм процедури МЖВ

    1. Обрати розв'язувальний елемент a[r][c]
    2. Якщо a[r][c] ≈ 0 → пропуск
    3. Зберегти копію таблиці
    4. a[r][c] ← 1
    5. Розв'язувальний стовпець (i ≠ r): a[i][c] ← −a_old[i][c]
    6. Розв'язувальний рядок (j ≠ c): залишається без змін
    7. Інші елементи (i ≠ r, j ≠ c): a[i][j] ← (a_old[i][j] · a_old[r][c] − a_old[i][c] · a_old[r][j])
    8. Всю таблицю поділити на a_old[r][c]
    9. Обміняти мітки: left[r] ↔ top[c]

### 2. Пошук оптимального розв'язку

    Пошук оптимального розв'язку (max Z)
    Початкова таблиця:
                    x1          x2          x3          x4          x5           b
    x6          2,0000      2,0000      1,0000      2,0000      1,0000      4,0000

    Z          -1,0000      0,0000      0,0000      0,0000     -3,0000      0,0000

    Всі b >= 0, опорний розв'язок є

    Крок 1
      Розв. стовпець: x5 (Z = -3,0000)
        x6: 4,0000 / 1,0000 = 4,0000
        x7: 4,0000 / 2,0000 = 2,0000
        x8: 9,0000 / 5,0000 = 1,8000
      Розв. рядок: x8 (min відношення b/a = 1,8000)
      Розв. елемент [2][4] = 5,0000
    Таблиця після кроку 1:
                    x1          x2          x3          x4          x8           b
    x6          1,4000      2,0000      0,8000      1,6000     -0,2000      2,2000
    x7          0,8000      1,0000     -0,4000      2,2000     -0,4000      0,4000
    x5          0,6000      0,0000      0,2000      0,4000      0,2000      1,8000
    Z           0,8000      0,0000      0,6000      1,2000      0,6000      5,4000

    Оптимальний розв'язок знайдено

      x6 = 2,2000

      x1 = 0
      x2 = 0
